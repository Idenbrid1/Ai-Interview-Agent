# Unit of Work 10: Navigation and Static Content

## Overview

This unit handles global navigation, static pages, dashboard functionality, and core user interface elements that provide structure and legal compliance for the platform.

## User Stories

### Global Navigation (Admin)

- **005_Admin Panel Side Menu**: As an admin, I want to access a side navigation menu with links to all administrative functions including logo/home, dashboard, user management, interview reports, interview preparation, message management, company management, announcements, user recommendations, settings, and logout

### Global Navigation (User)

- **005_User Panel Side Menu**: As a user, I want to access navigation appropriate for my user type (new graduate vs experienced) with links to top page, dashboard, interview reports, self-analysis, document creation, document drive, recommended companies, interview preparation, company messages, SPI testing, company analytics, friend referrals, profile, and settings

### Static Pages

- **024_Terms of Service**: As a user, I want to access the platform's terms of service to understand usage conditions
- **025_Privacy Policy**: As a user, I want to access the privacy policy to understand data handling practices
- **026_Company Information**: As a user, I want to access information about the operating company

### User Dashboard

- **Dashboard**: As a user, I want to view a comprehensive dashboard showing upcoming interviews, payment history, current subscription plan, contract renewal date, next billing date and amount, and administrator announcements

### Theme Management

- **Theme Switching**: As a user, I want to switch between job hunting and career change themes with appropriate background colors and menu adjustments

## Acceptance Criteria

### Admin Navigation

- Side menu provides access to all administrative functions
- Logo serves as home/dashboard link
- Navigation is consistent across all admin pages
- Logout functionality is easily accessible
- Current page is clearly indicated in navigation

### User Navigation

- Navigation adapts based on user type (new graduate vs experienced)
- Different menu items are shown based on user classification
- Navigation provides access to all relevant user functions
- Theme switching affects navigation appearance
- Current page is clearly indicated in navigation

### Static Content

- Terms of service page displays current legal terms
- Privacy policy page explains data collection and usage
- Company information page provides operating company details
- All static pages are easily accessible from footer or legal links
- Content is properly formatted and readable

### Dashboard Functionality

- User dashboard displays comprehensive overview including:
  - Upcoming interview schedule
  - Payment and subscription history
  - Current plan and renewal information
  - Next billing date and amount
  - Administrator announcements with popup details
- All dashboard elements link to relevant detailed pages
- Dashboard serves as central hub for user activities

### Theme and Customization

- Users can switch between job hunting and career change themes
- Theme changes affect background colors and menu styling
- Theme preference is saved and persists across sessions
- Different AI avatar configurations for different themes
